## 개요
코드잇 단기 심화에서 프로젝트를 진행하다가 Jest와 Cypress를 같이 사용해야 되는 일이 발생했습니다.
Jest를 사용하다가 Cypress를 설치해서 사용하다 보면 해당 오류가 나타나는것을 볼 수 있습니다.
```jsx
.toBeInTheDocument() // 'toBeInTheDocument' does not exist on type 'Assertion'
```
해당 오류는 타입에 관련된 오류고 사실 테스트를 실행하면 오류가 발생하지는 않습니다.
하지만 빨간줄은 항상 거슬리고 진짜 오류인지 아닌지 헷갈리게 만듭니다.
그래서 해결방법을 찾아보고 해결방안을 찾았습니다.
